# Hackathon Registration App

A comprehensive registration and check-in system for hackathons.

## Features

- **Admin Dashboard**: Import attendees via CSV, view statistics, manage registrations.
- **Registration Desk**: Search attendees by phone, register them for sessions or hackathon.
- **QR Check-in**: Generate QR codes for attendees and validate them at the door.
- **Email Notifications**: Send confirmation emails with QR codes.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express, Prisma, SQLite/PostgreSQL
